Site Plan: Ride Ready Moto

Site Name

Ride Ready Moto

I selected this name because it communicates that the site prepares new riders to be “ride ready” with basic motorcycle safety, learning tips, and confidence-building content.

Site Purpose

The purpose of this site is to help new motorcycle riders learn the basics of motorcycle safety, gear selection, riding techniques, and simple maintenance. The site will include:

  • Home page: Introduction & hero section
  • Learning and Tips page: Safety gear, riding basics, and maintenance
  • Contact page: Form for questions or joining a mailing list

Scenarios

  • “What gear do I need before riding a motorcycle for the first time?”
  • “How do I keep my motorcycle in safe condition with simple maintenance?”

Color Schema

Primary Color (Nav, Headings, Buttons):

#1E3A8A – Navy Blue

Accent Color (Call-to-action, highlights):

#E63946 – Red

Light gray is used for page background to improve readability.

Typography

  • Headings: Oswald (bold, used for page titles & section headers)
  • Body Text: Roboto (easy-to-read paragraph font)
